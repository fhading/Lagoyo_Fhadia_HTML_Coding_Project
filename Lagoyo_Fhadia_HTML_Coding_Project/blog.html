Blog Post 2:What is HTML?

HTML (Hypertext Markup Language) is a language used behind the scenes of every website. HTML uses tags to form the content in a visible way to display on a web page. It is relatively easy to learn and does not require lots of time to understand. It includes everything from your headers, spaces and paragraphs on your web page. HTML can be as simple as writing this blog, but it is written in specific tags so the browser can translate the content into a visible form. The browser does not display tags to the end user, but instead uses it to render the web page so the content is seen in an organised order.
